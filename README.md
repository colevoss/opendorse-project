# Baseball Stat Graphs by Cole Voss

## Getting Started

#### Clone this repo:

```
$ git clone git@github.com:colevoss/opendorse-project.git
```

#### Install dependencies

```
$ npm install
or
$ yarn install
```

#### Run application:

```
$ npm start
or
$ yarn start
```

#### Open application:

[localhost:3000](http://localhost:3000)
